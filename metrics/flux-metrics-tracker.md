# FLUX Metrics Tracker

Track weekly or per Pulse.

## 1) Pulse-to-Signal Rate
Definition: % of Pulses that produced measurable real-world change.

Formula:
`(Pulses with measurable change / Total Pulses) * 100`

## 2) Attractor Fidelity
Definition: Degree to which observed outcome matched Attractor definition.

Scoring rubric (0-5):
- 0 = no match
- 3 = partial match
- 5 = full match

## 3) Cycle Compression
Definition: Time-to-production reduction compared to baseline.

Formula:
`(Baseline cycle time - Current cycle time) / Baseline cycle time * 100`
